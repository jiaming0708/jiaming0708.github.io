Dotnet Core 在發布的時候預設會輸出 web.config，發佈到 IIS 伺服器如果有要調整 web.config 的內容，可以用以下幾種做法來達成。

關閉輸出

在 server 上修改，不要讓重新佈署的時候被覆蓋，可以使用 IsTransformWebConfigDisabled 來關閉輸出。
